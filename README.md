# Omflow

Backend for Omflow.yoga SPA

## Tech

RESTful JSON API
Node.js - express - mongoose

```
nodemon server/server.js
```

- [Local](http://localhost:3000)


## Heroku

- [Dashboard](https://omflow-svelte-bakckend.herokuapp.com/)

```
heroku login
heroku logs --tail -a omflow-svelte-backend
```
